# Psique

## Getting Started in development mode

### Back-end
```
cd /psique/web-app/web-api
```
run
```
npm run dev
```

### Front
```
cd /psique/web-app/web-console
```
run

```
npm start
```

### Prerequisites

What things you need to install the software and how to install them

web-api(Back-end) and web-console(Front-end):

node -v
